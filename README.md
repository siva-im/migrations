# Migration Tools

Repository migration tools for migrating to GitHub Enterprise Cloud.

## Supported Migrations

### Bitbucket Server to GitHub
- Complete migration toolkit
- Inventory collection scripts
- Post-migration validation

### Azure DevOps to GitHub  
- Migration scripts
- Repository inventory tools
- Validation utilities

## Structure

```
migrations/
├── bitbucket/          # Bitbucket Server migration
├── azure/              # Azure DevOps migration  
├── inventory/          # Inventory collection tools
└── validation/         # Post-migration validation
```
